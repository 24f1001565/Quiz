## Overview
LLM Analysis Quiz is an AI-driven system that automatically solves data-related quizzes using Python and LLMs. It handles data sourcing, preparation, analysis, and visualization, and submits answers to the provided endpoints.

The system is designed for students and developers to demonstrate automation, data processing, and AI integration skills.

## Key Features

| Feature | Description |
|---------|-------------|
| LLM-Powered Quiz Solver | Uses Python with headless browsers and AI techniques to extract, process, and submit answers. |
| Flexible Data Handling | Handles JSON, CSV, Excel, and other common data formats in quiz tasks. |
| Automated Submission | Validates secrets, fetches quiz pages, calculates answers, and submits them automatically. |
| Headless Browser Support | Uses Playwright to render JavaScript-heavy quiz pages. |
| Fast Execution | Optimized for quick response within the 3-minute quiz submission window. |
| Easy Deployment | Can run locally or on cloud platforms; secure secret verification implemented. |
